# autoscript — Коллекция Roblox скриптов

Универсальная коллекция Lua-скриптов для разных игр на Roblox.

## Скрипты по играм

| Файл | Игра |
|------|------|
| `AE.lua` | Anime Eternal |
| `BF.lua` | Blox Fruits |
| `BD.lua` | Blade Ball / другое |
| `ARX.lua` | ARX |
| `Rife.lua` | Rife |
| `RC.lua` | RC |

## Утилиты
- `AutoExe.lua` — авто-запуск скриптов
- `code.lua` — список промокодов для различных игр

## Дополнительно
- `SagaTestApiGConf.lua`, `SagaAnime/` — скрипты для Saga
- Используемые библиотеки: Obsidian, Wally, Luarmor

## Использование
Загрузить нужный `.lua` файл через Roblox executor.
